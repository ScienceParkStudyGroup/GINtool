GIN Tool user manual

 

For the installation of GINtool close Excel.

 

Prepare your file to make the BSU numbers compatible with those provided by the regulon file of Subtiwiki, which means an underscore after BSU (BSU_03500). For example by =IF(MID(A12,4,1)="_",A12,CONCATENATE(LEFT(A12,3),"_",RIGHT(A12,5))). The IF function is necessary because some newly annotated sRNAs contain already the underscore.

 

         Go to GIN tool (click GIN tool tab)

 

         Go to regulon file, klick select and find the Excel csv (!) file with the regulons from Subtiwiki (do not forget to select the csv file extension when searching for the file). It can be useful to make your own file and remove the SigA (housekeeping) regulon.[s1]  Go to operon file, klick select and find the Excel csv (!) file with defined operons. This file can be produced from the operon file that can be found in Subtiwiki.

 

         Click on load reference data.

 

         To determine the regulon, type of regulation and related gene ??in the Regulon csv file? [s2] your excel data file, go to the top right and select e.g. locus tag for bsu column, regulator for regulon, gene for gene[s3] [s4] , and mode for direction column.

 

         To indicate the direction of the mode of regulation (activation/repression) click on the red arrow regulation direction map. Select which mode descriptor in the Subtiwiki Excel csv file falls under up-regulated (activation) or down regulated (repression):

         Divide the FC range in high, middle and low, whereby FC under the low FC threshold are considered non-regulated. If you want to use one cutoff value, fill in the same value for all three ranges.

 

         To define relevance value select a cut-off for a p-value or related confidence value.[s5] 

 

         Your data should be organized in at least 3 columns, the first containing p-values, the second Fold Changes, and the third the BSU numbers. Select the data in these 3 columns (excluding headings) and click apply to range.

 

         The results are displayed over 3 sheets: Combined_1, Summary_1 and Mapped_1.[s6] 

 

         Mapped_1 sheet indicates to which regulons the genes belong. The colour red indicates that the regulator is a repressor and green indicates an activator.

 

         Summary_1 sheet summarizes the activity of regulons by indicating the number of genes regulated divided in the different fold categories. The genes that are displayed have been selected based on the indicated p-value or fold-change cut-off values.

o   The columns Total number in database and Total number in dataset indicate the total number of genes that are part of the regulon and that are found in the Regulon sheet from Subtiwiki and in the actual transcriptome dataset, respectively.

o   The columns Observed Counts and directions list the number of genes in the different fold-change categories.

o   The column Total Relevant is the sum of all genes that are above the low fold-change threshold level, and therefore considered to show a relevant change in expression.

o   The columns under Percentage indicate the number of genes as percentage of the Total number in dataset[s7] .

o   The columns under Logical direction indicate the change, expressed in percentage, that a regulon is activated or repressed. This is determined based on the mode of regulation of genes and the number of genes that show a Fold Change larger than the Low FC range threshold.

 

         The Combined_1 sheet integrates the data from the previous two sheets. It indicates the regulators that regulate the gene, the most likely activity of the regulator expressed in %, whereby an upwards or downwards pointing arrow indicates activation or repression, respectively[s8] . The next percentage shows the total (-tot) number of genes of the regulon that are regulated this way. For some regulators the mode (activation/repression) is unknown and in these cases the first percentage is not shown. Finally, the Combined_1 sheet also indicates to which operons the gene belongs.
